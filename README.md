# YelpCamp
This is a project where you can add campgrounds or review some of the existing campgrounds.
There's a map integrated to show the locations of all the campgrounds. All campgrounds and reviews can be deleted or updated by their author respectively. 
Basically the project demonstrate the CRUD operations and relation between different models. NodeJs is being used as backend and for database I have used MongoDB. 
The project has been deployed on heroku. LINK: https://yelpcamp-yelp.herokuapp.com
